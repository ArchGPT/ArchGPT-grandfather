<b>There are two ways to migrate data from Insomnia to Insomnium.</b>

1. In Insomnia, use the GUI button <kbd>Export files from the "Insomnia" Project</kbd> under `Preferences/Data` to export files (e.g. to your `~/Downloads`). <br/><br/>After that, open Insomnium and import these files using the <kbd >Import to the "Insomnium" Project</kbd> button under `Preferences/Data`.

2. Use the terminal directly. 

Linux:

```
cp -r ~/.config/Insomnia ~/.config/Insomnium
# or 
cp -r $XDG_CONFIG_HOME/Insomnia $XDG_CONFIG_HOME/Insomnium
```

MacOS:

```
cp -r ~/Library/Application\ Support/Insomnia/*.db ~/Library/Application\ Support/Insomnium/
```

Windows:

```
cp -r "%APPDATA%\Roaming\Insomnia" "%APPDATA%\Roaming\Insomnium"
```

Feel free to [open a discussion on github](https://github.com/ArchGPT/insomnium/discussions) if anything werid happens.

--- 

From Insomnia's documentation:

Insomnia stores data in the following location, depending on the platform:

- `%APPDATA%\Roaming\Insomnia` on Windows

- `$XDG_CONFIG_HOME/Insomnia` or `~/.config/Insomnia` on Linux

- `~/Library/Application\ Support/Insomnia` on macOS

(On Mac, you can open the parent folder by navigating to Help > Show App Data Folder in Insomnia)
